**Agentic RWA Treasury Manager**

Orbit is an autonomous AI agent that manages a treasury of Real-World Assets (RWAs) on the Arc L1 blockchain, utilizing Circle Wallets for custody, Stork for data, and Uniswap v4 for yield optimization.

## Repository Structure

- **`/agent`**: (Planned) Python-based AI Agent (LangGraph + FastAPI). The "Brain".
- **`/contracts`**: (Planned) Solidity Smart Contracts (Foundry). The "Vault" & "Hooks".
- **`/web`**: (Planned) Next.js Dashboard. The "View".
